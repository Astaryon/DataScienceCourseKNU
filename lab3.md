### Функція above(x, n), яка приймає вектор та число n, та повертає всі елементі вектору, які більше n. По замовчуванню n = 10
```
above <- function(a, n = 10)
{
  b = a[a > n]
  return(b)
}
above(arr <- c(1,2,11,12,13,5))

[1] 11 12 13

above(arr <- c(1,2,11,12,13,5),3)

[1] 11 12 13  5
```
